README

Name: MPyC (Multi Python Compiler)

License: ALOv1.1

About

MPyC is a multi-runtime Python compiler made for testing, running, and embedding Python in different environments. Supports packages, configs, and custom features.

Features

  • Custom Python Runtime
  • Manual Package Install
  • Support Offline Execution

Usage

  • cmd: "mpyc install [Package]" - Install a package inside runtime
  • cmd: "mpyc mod init" - Initialize a project and generate pmp.flag
  • cmd: "mpyc mod install [Package]" - Install packages inside MPyC mod system
  • cmd: "mpyc credit" - print the Credits
  • cmd: "C:\MPyC\PATH-ADD.bat" - add C:\MPyC\bin to env "Path" (Warning: env of system, not user)
